# WiFi-QR
This is an app to create a QR. If you scan this QR you will connect to the WiFi you've set.

-First step:

Change to the directory that consists your .csproj file

-Second step:

Type: dotnet add package QRCoder

-Third step:

Change the ssid & password

-Fourth step:

Check your .net9.0 dir in this project or the .net9.0/Debug to find the "wifi.png"
